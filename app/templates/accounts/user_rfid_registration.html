Pendaftaran User & Kartu RFID

Halaman ini langsung memakai endpoint yang sudah Anda buat, yaitu /api/accounts/register-rfid/ dan /api/accounts/create-user-rfid/.

Daftarkan Kartu ke User Existing

Gunakan form ini jika akun user di auth_user sudah ada, lalu Anda hanya ingin menempelkan kartu RFID ke user tersebut.

Buat User Baru + Kartu RFID

Gunakan form ini jika user belum ada. Sistem akan membuat data di auth_user dan accounts_userprofile sekaligus.

Catatan: untuk form user existing, isi salah satu User ID atau Username. Tidak harus dua-duanya.
